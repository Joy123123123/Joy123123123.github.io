# amznon.me — Personal Website

> Personal portfolio and website for **Md Jamil Islam** ([@Joy123123123](https://github.com/Joy123123123)), hosted at [amznon.me](https://amznon.me) via GitHub Pages.

## 🚀 Stack

- **Pure HTML5 / CSS3 / Vanilla JS** — zero dependencies, zero build step
- **GitHub Pages** — automatic deployment from the `main` branch
- **Custom domain** — `amznon.me` via `CNAME`

## 📁 Structure

```
.
├── index.html          # Main page
├── CNAME               # Custom domain (amznon.me)
├── assets/
│   ├── css/
│   │   └── style.css   # All styles
│   └── js/
│       └── main.js     # Interactivity (navbar, scroll reveal, counters)
└── README.md
```

## ✨ Features

- Responsive, mobile-first design
- Dark theme with accent colors
- Smooth scroll navigation with active link tracking
- Animated statistics counters
- Scroll reveal animations
- Interactive 3D code card
- Hamburger menu for mobile

## 🌐 Deployment

Push to `main` → GitHub Pages automatically deploys to `amznon.me`.

No build process required — everything is static HTML/CSS/JS.
